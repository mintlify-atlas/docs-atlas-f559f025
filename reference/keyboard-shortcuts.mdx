title: "KillerPDF keyboard shortcuts: complete reference"
description: "Complete list of keyboard shortcuts in KillerPDF covering file operations, editing tools, text selection, search, print, undo, and annotation management."
---

KillerPDF provides keyboard shortcuts for the most common actions. Shortcuts are active whenever no text box is focused — when you are typing in a floating text box or editing inline text, keys are passed directly to that input instead.

## File operations

| Shortcut | Action |
| --- | --- |
| `Ctrl+P` | Print the current PDF (annotations are flattened into the output) |

## Editing

| Shortcut | Action |
| --- | --- |
| `Ctrl+Z` | Undo the last annotation added to the current page |
| `Delete` | Delete the currently selected annotation |

## Text

| Shortcut | Action |
| --- | --- |
| `Ctrl+A` | Select all text on the current page and copy it to the clipboard |
| `Ctrl+C` | Copy the current drag-selected text to the clipboard |
| `Enter` | Commit the active text box or inline text edit |
| `Shift+Enter` | Insert a new line inside a floating text box (Text tool) |
| `Escape` | Discard the active text box or cancel an inline text edit |

## Navigation and search

| Shortcut | Action |
| --- | --- |
| `Ctrl+F` | Toggle the search bar open or closed |
| `Enter` | Run the search when the search box is focused |
| `Escape` | Close the search bar when it is open |

<Tip>
  Shortcuts that conflict with an active text input are suppressed automatically. For example, pressing `Escape` while editing text discards the edit, and pressing `Enter` commits it — neither key closes the search bar while a text box is focused.
</Tip>

## Context menu

Right-clicking on the page canvas opens a context menu that exposes the same actions with their keyboard shortcut hints displayed inline. You can switch tools, copy text, delete or undo annotations, and clear all annotations on the page from the context menu without using the toolbar.
